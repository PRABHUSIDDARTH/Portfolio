02 — Projects

Things I've built
that actually work.

01
Swara — Offline Music Player
A fully offline Android music player built in Java. Royal purple & gold custom theme, MediaPlayer-based playback in a Bound Service, RecyclerView song list, SeekBar with Handler/Runnable, AudioFocus, and a polished custom PlayerActivity. No internet. No ads. Pure local music.
Java Android Studio MediaPlayer API Bound Service AudioFocus
⬡ GitHub
02
Self-Hosted AI Assistant (Raspberry Pi 5)
Built a private AI assistant running locally on a Raspberry Pi 5 — Ollama with Mistral 7B, Java Spring Boot backend, PostgreSQL, Nginx reverse proxy, and a dark-themed chat UI with markdown rendering. 100% offline and self-hosted.
Raspberry Pi 5 Ollama / Mistral 7B Spring Boot PostgreSQL Nginx
⬡ GitHub
03
Safety Monitoring System (Jetson Nano)
Real-time helmet and face mask compliance detection using Jetson Nano P3450, Raspberry Pi 5, and Intel RealSense D435. TensorRT-optimized model, SQLite logging, Flask web dashboard, and a cooldown-based violation tracker.
Jetson Nano TensorRT Intel RealSense Flask SQLite Python
⬡ GitHub
04
Distraction Monitor AI
An AI-powered tool to detect and monitor user distraction in real-time. Built as part of early explorations into CV-based productivity tools.
Python Computer Vision AI/ML
⬡ GitHub
05
Task Management System
A Java-based task management system demonstrating core OOP principles, persistence, and CRUD workflows — one of the foundational projects in the Java learning journey.
Java OOP SQL
⬡ GitHub
